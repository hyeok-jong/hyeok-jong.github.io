1. 등기사항전부 증명서 (등기부등본)#

참고 영상

  • 표제부 - 집 주에 대한 정보
  • 갑구 - 소자 대한 정보
  • 을구 - 소유권 이외 주요정보

How to check#

  1. 열람일시
  2. 표제부의 주소 == 계약서 주소
  3. 갑구의 임대인 인적사항 == 계약서 인적사항 == 신분증 == 계좌주인
  4. If one of [가압류, 압류, 가등기, 가처분] is in 갑구 => X
  5. 을구 - 근저당권 : 해당 집을 담보로 대출 받은 사항 (채권최고액 == 대출금액, 당연히 적을수록 좋음)
  6. 임차권등기명령 is in 을구 => X
